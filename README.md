## Overview
Schedule Helper is a cross-platform desktop application (built with .NET and WPF) that keeps your personal and team schedules in one place. The project focuses on quick capture, clear visibility of priorities, and gentle nudges to stay on track.

## Key Highlights
- **Unified task board:** Capture daily todos, recurring chores, and work items with lightweight categorization.
- **Timeline clarity:** Visualize upcoming commitments with calendar-style views and color-coded priorities.
- **Progress at a glance:** Track completion streaks and overdue items to spot bottlenecks early.
- **Friendly reminders:** Configure notifications and reminders tailored to your working hours.
- **Export-friendly:** Plan to support printable summaries and shareable snapshots for teammates.

## Project Status
This repository is in active setup. Core scheduling flows, UI polish, and integrations are being fleshed out. Contributions and feedback are welcome as the experience evolves.

## Getting Started
1. Ensure you have the .NET SDK installed (version 8.0 or later recommended).
2. Open the solution file at `src/SheduleHelper.sln` in your preferred IDE (Visual Studio or JetBrains Rider).
3. Restore dependencies and run the WPF app project to explore the current UI.
